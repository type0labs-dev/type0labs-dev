# Mauricio Juba — TYPE:ZERO

**Design engineer.** I build the systems that keep AI-generated software coherent: design systems as LLM context, and agents deployed into real operations.

Fifteen years in tech, ten shipping enterprise design at **Dell** and **IBM**: design tokens across dell.com, GenAI product frameworks, design systems for 8M+ customers. Today I also ship the code.

**Open source**

- 🎨 **[ouroboros-ui](https://github.com/Type-zero-labs/ouroboros-ui)**: shadcn/ui rebuilt natively in egui/Rust. 71 components, token-first, with governance tests that **fail the build** when a raw value sneaks past the tokens. MIT, [docs + live storybook](https://ouroboros-ui.typezerolabs.com).
- 🧬 **[ui-morph](https://github.com/type0labs-dev/ui-morph)**: a generative-UI engine that keeps an LLM constrained to your design tokens, so the layout adapts per persona without going off-system.
- 🔒 **[actual-mcp](https://github.com/type0labs-dev/actual-mcp)**: an MCP server with a per-tool safety layer, so an agent with write access to real data can't quietly delete anything.
- 🐍 **Ouroboros**: MMO engine and authoring IDE in Rust. 120k LOC, spec-driven, QUIC networking, `bevy_ecs` server, `wgpu` client.
